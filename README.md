# 🐙 롱숏 신탁 (LongShort Oracle)

버튼을 누르면 점쟁이 문어가 BTC 선물 **롱/숏 중 하나를 "신탁"으로 내려주는** 오락 사이트.

실제 시장 지표 5종(공포탐욕지수·펀딩비·탑 트레이더 포지션·전체 계정 롱숏 비율·24h 모멘텀)으로
확률을 살짝 기울이고, **그 오차 범위(15%~85%) 안에서만 랜덤**으로 결과를 뽑는다.
결과에는 수치·목표가를 절대 노출하지 않는다 — 이것은 예측 도구가 아니라 오락이다.

전체 명세: [`01-claude-code-프롬프트.md`](./01-claude-code-프롬프트.md) · 배포 방법: [`DEPLOY.md`](./DEPLOY.md)

## 기술 스택

- Next.js (App Router) + TypeScript + Tailwind CSS v4
- DB 없음, 로그인 없음, API 키 없음 (Phase 1)
- 배포: Vercel 무료 플랜

## ⚠️ Vercel 리전 — icn1 고정 (치명적 함정)

**Binance API는 미국 IP의 요청을 HTTP 451로 차단한다.**
Vercel 함수의 기본 리전은 미국(iad1)이므로, 그대로 배포하면 지표 5개 중 4개(바이낸스 소스)가
전부 실패해 사이트가 항상 "운빨 모드"로만 동작한다.

이를 막기 위해 [`vercel.json`](./vercel.json)에 서울 리전을 고정해 두었다:

```json
{ "regions": ["icn1"] }
```

이 파일을 지우거나 리전을 바꾸면 안 된다.

## 프로젝트 구조

```
app/                페이지·라우트
  api/market/       지표 5종 API (서버 전용, 60초 캐시)
  result/[side]/    공유용 결과 페이지 + 동적 OG 이미지
  guide/[slug]/     가이드 글 (마크다운 렌더링)
lib/oracle.ts       신탁 알고리즘 (순수 함수, 스펙 고정)
lib/market.ts       외부 지표 수집 (개별 3초 타임아웃, 장애 내성)
config/oracle.config.ts  가중치·상수
config/ments.ts     재미 멘트 풀 (롱/숏 각 24개) + 로딩 문구
content/guide/*.md  가이드 글 10개 (애드센스 심사용)
components/AdSlot.tsx    광고 자리 예약 (고정 높이 → CLS 방지)
```

## 절대 규칙 (스펙 7장)

- 결과에 수치·목표가·진입가 노출 금지 — 오직 "롱"/"숏" + 재미 멘트
- "시그널·자문·추천 진입" 등 금융 자문 어휘 금지 → "신탁", "점괘"만
- 클라이언트가 외부 API 직접 호출 금지 (반드시 `/api/market` 경유)
- 신탁 버튼 근처 광고 슬롯 배치 금지
- 지표 전부 실패해도 버튼은 항상 동작 (P=0.5 운빨 모드)

## 면책

본 사이트는 재미를 위한 오락 서비스이며 어떠한 금융 상품의 매매 권유·투자 자문도
제공하지 않습니다. 선물·레버리지 거래는 원금 전액 손실 위험이 있습니다.
