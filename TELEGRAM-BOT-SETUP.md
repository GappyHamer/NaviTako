# 텔레그램 문의 봇 설정 가이드

방문자가 `/contact` 페이지에서 문의를 남기면, 텔레그램 봇이 운영자(나)에게
바로 메시지를 보내주는 기능입니다. 아래 순서대로 5분이면 연결됩니다.

준비물은 텔레그램 계정 하나뿐이고, 환경변수 두 개(`TELEGRAM_BOT_TOKEN`,
`TELEGRAM_CONTACT_CHAT_ID`)를 Vercel에 넣는 것으로 끝납니다.
두 값 중 하나라도 비어 있으면 기능은 자동으로 비활성화되고, 폼은 "잠시 후
다시 시도" 안내만 보여줍니다(사이트는 정상 동작).

---

## 1) 봇 만들고 토큰 받기 (`TELEGRAM_BOT_TOKEN`)

1. 텔레그램에서 **@BotFather** 를 검색해 대화를 엽니다.
2. `/newbot` 을 보냅니다.
3. 봇의 **표시 이름**(예: `타코쨩 문의봇`)을 정합니다.
4. 봇의 **유저네임**을 정합니다. 반드시 `bot` 으로 끝나야 합니다
   (예: `ttakochan_contact_bot`).
5. 그러면 BotFather가 **봇 토큰**을 줍니다. 이렇게 생겼습니다:

   ```
   123456789:AAErk...abcDEF
   ```

   이 값이 `TELEGRAM_BOT_TOKEN` 입니다. **비밀번호처럼 다루세요.**

---

## 2) 내 봇에게 먼저 말 걸기 (필수)

텔레그램 봇은 **상대가 먼저 대화를 시작하지 않으면 메시지를 보낼 수 없습니다.**
그래서 방금 만든 봇에게 내가 먼저 인사를 해줘야 합니다.

1. 텔레그램 검색창에 방금 정한 봇 **유저네임**(예: `@ttakochan_contact_bot`)을
   검색합니다.
2. 대화를 열고 **시작(Start)** 버튼을 누릅니다.
3. 아무 메시지나 하나 보냅니다 (예: `hi`).

---

## 3) 내 chat_id 얻기 (`TELEGRAM_CONTACT_CHAT_ID`)

문의가 도착할 **내 채팅방 ID**를 알아냅니다.

**방법 A — getUpdates (권장)**

1. 브라우저 주소창에 아래 주소를 열되, `<토큰>` 자리에 1단계에서 받은
   봇 토큰을 그대로 붙여넣습니다:

   ```
   https://api.telegram.org/bot<토큰>/getUpdates
   ```

   예: `https://api.telegram.org/bot123456789:AAErk...abcDEF/getUpdates`

2. 응답 JSON에서 `"chat":{"id": ...}` 부분의 숫자를 찾습니다:

   ```json
   "message": { "chat": { "id": 987654321, "first_name": "..." } }
   ```

   이 숫자(`987654321`)가 `TELEGRAM_CONTACT_CHAT_ID` 입니다.

   > 결과가 비어 있으면(`"result":[]`) 2단계에서 봇에게 메시지를 아직
   > 안 보낸 것입니다. 메시지를 하나 보내고 페이지를 새로고침하세요.

**방법 B — @userinfobot**

텔레그램에서 **@userinfobot** 에게 말을 걸면 내 숫자 id를 바로 알려줍니다.

> ⚠️ 보안 주의: getUpdates 주소에는 봇 토큰이 그대로 들어갑니다. id를
> 확인했으면 그 브라우저 탭을 닫으세요. 토큰은 절대 공개된 곳(깃허브,
> 채팅 등)에 붙여넣지 마세요.

---

## 4) Vercel에 환경변수 넣고 재배포

1. Vercel 대시보드 → 해당 프로젝트 → **Settings** → **Environment Variables**.
2. 아래 두 개를 **Production** 환경으로 추가합니다:

   | Name | Value |
   | --- | --- |
   | `TELEGRAM_BOT_TOKEN` | 1단계의 봇 토큰 |
   | `TELEGRAM_CONTACT_CHAT_ID` | 3단계의 chat_id 숫자 |

3. 저장한 뒤 **Deployments** 탭에서 최신 배포를 **Redeploy** 합니다
   (환경변수는 재배포해야 반영됩니다).

---

## 5) 테스트

1. 배포된 사이트의 `/contact` 페이지를 엽니다.
2. 문의 내용을 적고 **문의 보내기**를 누릅니다.
3. 텔레그램에 `📮 새 문의 · 타코쨩` 형식의 메시지가 도착하면 성공입니다.

---

## 문제 해결

- **폼이 "잠시 후 다시 시도" 만 뜬다** → 환경변수 두 개가 제대로 들어갔는지,
  재배포했는지 확인하세요(둘 중 하나라도 없으면 기능이 꺼집니다).
- **텔레그램으로 안 온다** → chat_id가 정확한지, 2단계에서 봇에게 먼저
  말을 걸었는지 확인하세요.
- **토큰이 노출된 것 같다** → BotFather에서 `/revoke` 로 토큰을 폐기하고
  새 토큰을 발급받아 Vercel 환경변수를 교체하세요.
